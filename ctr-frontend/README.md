# CTR Frontend (React + Vite)

This frontend now consumes a real backend API instead of `localStorage` mocks.

## Setup

1. Install dependencies:
   - `npm install`
2. Set API URL (optional, default shown):
   - `VITE_API_BASE_URL=http://localhost:8080`
3. Start dev server:
   - `npm run dev`

## Backend dependency

Run `ctr-backend` first (see [backend README](C:/CTR/ctr-backend/README.md)).
